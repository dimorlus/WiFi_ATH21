Есть устройство на ESP8266 которое подключается к MQTT брокеру и посылает такие данные:

ORLOV/HUMT/HT_3C71BF29A68E/HUMT:SNTP: 1755580662, TEMP: 26.4 C, HUM: 56.3 %, DEW: 17.0 C

Я хочу написать программу на C/С++ (две программы) для Windows и Linux, которая бы подключалась к 
MQTT брокеру, подписывалась на все топики вида ORLOV/HUMT/HT_3C71BF29A68E/HUMT 
(HT_3C71BF29A68E - для каждого устройства уникален, это его MAC) получало данные и записывало 
их в текстовый файл с именем, совпадающем с именем устройства и расширением .csv в соответствии
с заданным регулярным выражением Search и Replace. Первой строкой должен идти Header. Настройки 
программы должны храниться в .ini файле, совпадающем по имени с именем программы (где его помещать 
для Windows и Linux - отдельный вопрос).  

[MQTT]
URL=dorlov.no-ip.com:1883
user=dimorlus
passwd=dorlov
topic_base=ORLOV

[DATA]
Header = SNTP; TEMP; HUM
Search = "SNTP: ([0-9]+), TEMP: ([0-9\.]+) C, HUM: ([0-9\.]+) %, DEW: ([0-9\.]+) C"
Replace = "\1;\2;\3

[LOG]
;Folder="%APPDATA%\mqtt_Logger\data"
Folder="d:\Projects\LED\Alex\WiFi_AHT21\PC\LOG"
